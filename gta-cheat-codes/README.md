# GTA Cheat Codes — PWA

The ultimate offline GTA cheat code encyclopedia. 100 cheats across 9 games for PC, PlayStation, and Xbox. Fully offline, installable, Microsoft Store ready.

---

## 🛠 Prerequisites

- Node.js 18+
- npm 9+

---

## 📦 Setup

```bash
# 1. Install dependencies
npm install

# 2. Run development server
npm run dev
# Opens at http://localhost:3000

# 3. Production build
npm run build
npm start
```

## 📦 Package for Microsoft Store (PWABuilder)

1. Deploy to Vercel and get your live URL (e.g. `https://gta-cheat-codes.vercel.app`)
2. Go to **https://www.pwabuilder.com**
3. Enter your Vercel URL → click Start
4. Click **Package for Store** → choose **Windows (MSIX)**
5. Fill in publisher details, download the `.msix` package
6. Submit to **Microsoft Partner Center** at https://partner.microsoft.com

> **Note:** PWABuilder requires a valid HTTPS URL, correct manifest.json, and working service worker. All three are included in this project.
